- 内置数据类型
  八种基本类型：六种数字类型，一种字符类型，一种布尔型
  1. byte  8位 默认值 0
  2. short  16位  默认值 0 
  3. int  32位  默认值 0
  4. long  64位  默认值 0L
  5. float  32位  默认值0.0f
  6. double  64位  默认值0.0d
  7. char  16位Unicode字符
  8. boolean  一位  默认值false
- 引用数据类型
  引用类型指向一个对象，指向对象的变量是引用变量。
  对象、数组都是引用数据类型。
  所有引用类型的默认值都是null
